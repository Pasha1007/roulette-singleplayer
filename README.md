# swipegame-frameworks   Roulette


## 1. Introduction to Node.js Environment

### 1.1. Install Node.js version 19.8.1 or above
### 1.2. Install npm version 9.5.1 or above


## 2. Install dependencies

### 2.1. Install dependencies
```
  npm install
``` 
### 2.2. Run the project
```
  npm run start
```
### 2.3. Package and release the project
```
  npm run build
```
